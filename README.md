# Shopify Dashboard - Meteor

Een moderne dashboard applicatie voor e-commerce project management met Shopify integraties.

## Features

- **Sales Calculator**: Interactieve tabel voor project kostencalculatie
- **Project Management**: Overzicht van alle actieve projecten
- **Theme Gallery**: Shopify theme collectie en beheer
- **App Management**: Shopify app integraties
- **Settings**: Gebruiker en applicatie instellingen
- **Knowledge Quiz**: Shopify kennis test

## Technologie Stack

- React 18
- Tailwind CSS
- Lucide React Icons
- Modern gradient design

## Installatie

1. Installeer dependencies:
```bash
npm install
```

2. Start de development server:
```bash
npm start
```

3. Open [http://localhost:3000](http://localhost:3000) in je browser.

## Gebruik

### Sales Calculator
- Vink taken aan/uit om ze in de offerte op te nemen
- Pas uren en tarieven aan voor realtime berekening
- Genereer offertes met de knoppen onderaan

### Project Management
- Bekijk project voortgang en deadlines
- Beheer team toewijzingen
- Track project status

### Integraties (Toekomstig)
- Teams integratie voor communicatie
- Confluence voor documentatie
- Jira voor issue tracking
